# 增强现实乐器 / 合成器 / synthesizer

本仓库仅用于部署 AR 乐器公开演示页面。

本演示使用两套识别系统：平台入口码用于微信扫码进入公开演示页；乐器卡不含二维码，只作为页面内相机识别的整张图像目标。用户进入页面后，对准合成器或鼓机乐器卡，系统识别对应 pattern 后显示对应 AR 乐器。

AR 合成器默认尺寸固定，不随屏幕大小、卡片远近或可视区域自动缩放。用户可以通过双指手动缩放，缩放后的合成器仍然跟随乐器卡锚点。

卡片二维码不链接 GitHub 仓库。当前二维码写入的是 GitHub Pages 演示入口页：

```text
https://diandianking.github.io/ar-synyher/
```

该 Pages 地址直接进入 AR 演示页。`/synth/` 入口路径仍保留为兼容入口，但当前卡片优先使用更短、更容易扫码的根地址。
